# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这个项目收集了常用的 Claude Code skills。

## 项目要求

- 对话和文档内容以简体中文为主
- 保存的文件名使用英文，格式为：`this_is_a_file_name.filetype`
- 创建的新文件在标题下增加 Created At，如：`Created At: Jan 8, 2026`
- 更新已经存在的文件在 Created At 下增加或更新 Updated At，如：`Updated At: Jan 8, 2026`
